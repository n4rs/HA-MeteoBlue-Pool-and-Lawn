# Contribution guidelines

This repository is maintained by @n4rs as **HomeAssistant Pool and Lawn**.

## Issues and feature requests

Use GitHub issues in this repository to report bugs or request features:

https://github.com/n4rs/HA-MeteoBlue-Pool-and-Lawn/issues

## Pull requests

Pull requests may be reviewed at the maintainer's discretion. By submitting a
pull request, you agree that your contribution may be incorporated into this
project under the repository's composite license, including the proprietary
terms that apply to HomeAssistant Pool and Lawn modifications and additions.

## Development

Use the project scripts where possible:

- `scripts/setup` to install dependencies with `uv sync --frozen`.
- `scripts/test` to run the test suite.
- `scripts/lint` to format and lint the code.
